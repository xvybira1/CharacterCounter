# CharacterCounter

Simple CLI tool that counts characters in a string. Done locally. No more searching on internet to perform such a simple task.

# Installation

Clone the repository. Compile the code with `gcc charcounter.c -o charcounter`

# Usage

`charcounter -h` or `charcounter --help` for help.

`charcounter 123456789` or `charcounter -c 1 111111111`
